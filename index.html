Kitbase install

Swift Package Manager

You can also use Swift Package Manager to add KitBase as a dependency:

  1. In Xcode, go to File > Swift Packages > Add Package Dependency.
  2. Enter the repository URL: https://github.com/sinbadbd/KitBase.git
  3. Specify the version or branch you want to use.

Manual Installation

If you prefer not to use any dependency manager, you can manually add the KitBase source files to your project:

  1. Download the latest release or clone the repository.
  2. Drag and drop the KitBase folder into your Xcode project.

Usage

After installing KitBase, you can start using it in your SwiftUI project. Here's a quick example:

        
import SwiftUI
import KitBase

struct ContentView: View {
    var body: some View {
        Button("Custom Button") {
            // Handle button tap
        }
        .buttonStyle(KitBaseButtonStyleBuilder()
            .setBackgroundColor(.blue)
            .setForegroundColor(.white)
            .build()
        )
    }
}

KitBaseButtonStyleBuilder Summary

The KitBaseButtonStyleBuilder struct is designed to build custom button styles with various configurable parameters. The ContentBuilderView SwiftUI view showcases how to use this builder to create a button with specific style configurations.

Please note that the actual implementation details within the build() method and the usage in ContentBuilderView are not provided in this summary. If you need further details or have specific questions about any part of the code, feel free to ask!

             
struct ContentBuilderView: View {
    var body: some View {
        VStack(spacing: 16) {
            Button("icon with button", action: {
                print("")
            })
            .buttonStyle(
                KitBaseButtonStyleBuilder()
                    .setBackgroundColor(.gray)
                    .setForegroundColor(.black)
                    .setButtonWidth(200)
                    .setButtonHeight(30)
                    .setIcon("square.and.arrow.up")
                    .setImage("ic_edit")
                    .setIconColor(.red)
                    .setIconColor(.red)
                    .setIconWidth(20)
                    .setIconHeight(20)
                    .setShowShadow(false) // Default true
                    //.setPaddingAll(paddingAll: 0)
                    .setPaddingVertical(paddingVertical: 0)
                    .setPaddingHorizontal(paddingHorizontal: 0)
                    .build()
            )
        }
    }
}
Parameter
Method Parameter Value (example) Description
KitBaseButtonStyleBuilder setBackgroundColor .gray The background color of the button.
setForegroundColor .black The text color of the button.
setButtonWidth 200 The width of the button.
setButtonHeight 30 The height of the button.
setIcon "square.and.arrow.up" The system icon to be displayed on the button.
setImage "ic_edit" The image name to be used on the button.
setIconColor .red The color of the icon.
setIconWidth 20 The width of the icon.
setIconHeight 20 The height of the icon.
setShowShadow false Whether to show a shadow around the button. Default is true.
setPaddingVertical 0 The vertical padding of the button.
setPaddingHorizontal 0 The horizontal padding of the button.
setPaddingAll 0 The padding applied to all sides of the button.
build Builds and returns a KitBaseButtonStyle instance with the configured parameters.
